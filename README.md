## What is Countio
Countio is simple WatchOS App made with Swiftly SwiftUI. You can count anything quickly with Countio!!!

## Features
* Count Anything Quickly
* Increase ? Decrease
* 6 Options To Count
* Switch Quickly +/-
* Remembers Every Last Count

## Requirements
* Xcode 11
* SwiftUI
